## Html Text Formatting

> * b : Bold
> * i : Italic
> * small : Smaller text
> * u : Underline
> * mark : HighLight Text
> * important : Underline
> * em : emphasized text
> * strong : Important Text
> * Sub : SubScript
> * Sup : SuperScript
> * del : Deleted Text
> * ins : inserted text
